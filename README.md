Project Title: JSON-Based Product Display Web App

This project is a simple and dynamic web application built using HTML, CSS, and JavaScript that demonstrates how to create and manage a product listing using a JSON file as a data source.

The application fetches product data from a local JSON file and dynamically renders it on the webpage using JavaScript. It helps beginners understand how to work with APIs, handle JSON data, and manipulate the DOM effectively.


🚀 Features:
📂 Fetch data from a JSON file
🛒 Dynamic product listing
💻 Clean and responsive UI using CSS
🔄 Real-time DOM manipulation
⚡ Lightweight and fast performance
📱 Mobile-friendly design
